# For Setting Up Environment

To Activate :
```
conda activate
```

To Deactivate : 
```
conda deactivate
```

For Listing Environemnt :
```
conda env list
```

To Create a Virtual Env :
```
conda create --name {ENV-NAME} python=3.9
```
Note: *Pyhton version may vary for your system*
